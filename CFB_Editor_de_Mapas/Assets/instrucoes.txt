INSTRUÇÕES:
Clique e arraste com o botão esquerdo do mouse para selecionar.
Com o direito, no fundo para mover a câmera, ou entre nós para fazer ou desfazer conexões.
Segure [CTRL] e clique para criar ou destruir nós.
[G] para mover seleções.
[R] para rotacionar seleções.
[S] para escalar seleções.
[D] para duplicar seleções.
[X] ou [DEL] para deletar seleções.
[C] para cortar todas as conexões entre a seleção e o resto do mapa.
[A] para formar conexões entre nós próximos na seleção.
[F] para aplicar forças de atração/repulsão entre os nós da seleção.
[L] para selecionar todos os nós conectados com a seleção atual.
[N] para gerar um novo pedaço de mapa. (aperte enquanto faz um anel de seleção para determinar o tamanho do pedaço gerado)
[CTRL+Z] para desfazer (transformações apenas)
[CTRL+Y] para refazer (transformações apenas)
